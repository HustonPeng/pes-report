# Pes-Report Tool

## How to use?

* Install. Java8, Maven
* Prepare Execl: Two columns + One Sheet.
  One is name, another is receipt no.
* Execute: 
```shell
java -jar ./pesReport.jar ${excelPath} ${templatePath} ${outputDir}
```

## Limitation

* Generate a template only once.
* Should 33 names for lighting cards.
